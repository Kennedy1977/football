# v0.0.65 - Auto-pick and auto-save lineup when formation changes

Date: 2026-03-11
Type: feat
Scope: squad

## Summary

- Updated squad management so changing formation immediately auto-picks the best eligible XI for that formation and valid lineup changes are auto-saved without requiring a manual save click.

## Added

- Added debounced auto-save behavior for lineup/formation changes in squad management.

## Changed

- Formation change in squad now triggers best-XI auto-pick for the target formation (with bench recalculated from remaining top players).
- Manual save dependency removed from normal lineup editing flow; save now happens automatically when lineup is valid and changed.
- Auto-pick button now aligns with auto-save behavior and shows save-in-progress feedback.

## Fixed

- Fixed mismatch where formation could change without reselecting the best-fit XI for the new shape.

## Removed

- None.
