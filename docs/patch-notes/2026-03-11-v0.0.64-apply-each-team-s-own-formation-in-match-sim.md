# v0.0.64 - Apply each team's own formation in match sim

Date: 2026-03-11
Type: fix
Scope: match-live

## Summary

- Fixed live-match formation handling so each side uses its own lineup formation (for example, home `4-2-4` vs away `4-3-3`) instead of both teams defaulting to a shared shape.

## Added

- Added formation fields to `StartMatchResponse` and match prep state for both teams.
- Added formation display row in match prep (`your formation` vs `opponent formation`).

## Changed

- `/api/match/start` now reads `lineups.formation_code` for both clubs and returns normalized formations.
- Live match runtime config now passes `homeTeam.formation` and `awayTeam.formation` into Phaser.
- Phaser team placement now resolves formation-specific coordinate layouts for all supported formations (`4-4-2`, `4-3-3`, `4-5-1`, `4-2-3-1`, `3-5-2`, `5-3-2`, `4-2-4`).

## Fixed

- Fixed a bug where the simulation always used a static `4-4-2`-style shape regardless of selected formations.

## Removed

- None.
