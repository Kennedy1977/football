# v0.0.73 - add emergent support-triangle behavior in live sim

Date: 2026-03-12
Type: feat
Scope: match-sim

## Summary

- Updated the live match ambient AI so support triangles emerge from scoring rules instead of fixed support pairs.
- Ball carriers now receive a safer recycle outlet and a progressive outlet at different angles/depths, with dynamic width support when available.

## Added

- Support-shape planner for ambient possession (`SAFE`, `PROGRESSIVE`, `WIDTH` roles).
- Passing-lane quality and local defensive pressure scoring for off-ball support evaluation.
- Third-man link value in progressive support scoring to encourage bounce/layoff sequences.

## Changed

- Replaced static `pickSupportPair` usage in ambient phases with dynamic support planning based on pressure, zone, spacing, and lane access.
- Updated ambient action scoring to consider support shape quality and progression threat.
- Updated defender movement in passing/dribbling phases to contest easy outlets and progressive lanes.
- Recycle behavior now prefers an actual safety outlet receiver instead of repeatedly carrying backwards with the same player.

## Fixed

- Reduced isolated-ball-carrier moments by maintaining at least two viable local passing outlets more consistently.
- Reduced flat/stacked support lines via angle-separation and depth-variation penalties in support pair selection.

## Removed

- None.
