# v0.0.76 - add manager home-away kit picker with color lock

Date: 2026-03-12
Type: feat
Scope: profile

## Summary

- Added manager-level home/away kit configuration on the profile screen using shirt sprite selections.
- Enforced a hard rule that home and away shirts cannot be the same color in UI and API validation.
- Wired selected kit colors into match-live sim rendering with away-kit fallback when team colors clash.

## Added

- New `PUT /onboarding/club/kits` endpoint to update `home_kit_json` and `away_kit_json` for the current manager club.
- New profile UI section for `Team Kits` with selectable shirt options (red, blue, yellow, green, white, black) sourced from `shirts-sprite.webp`.
- New API contracts and RTK mutation for kit updates:
  - `UpdateClubKitsRequest`
  - `UpdateClubKitsResponse`
  - `useUpdateClubKitsMutation`

## Changed

- Dashboard summary now returns `club.homeKit` and `club.awayKit` so profile kit choices can load current saved values.
- Match live page now resolves kit colors robustly and avoids same-color clashes by preferring Team B away kit when needed.

## Fixed

- Resolved missing helper compile break in `match/live/page.tsx` by adding simulation kit color resolution utilities.

- None.
