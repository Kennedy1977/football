# v0.0.60 - Add walk-on sequences and compact commit message enforcement

Date: 2026-03-11
Type: feat
Scope: match-sim

## Summary

- Added cinematic team transitions in the live match simulation and relaxed commit-message enforcement to a shorter format while keeping patch-note traceability.

## Added

- Match intro/halftime/full-time transition choreography:
  - both teams walk on from the left before kickoff
  - both teams walk off left at halftime, then return and retake positions
  - at full-time, losing team walks off left and winning team celebrates

## Changed

- Striker decision weighting in ambient play now favors direct attacking actions in advanced zones and reduces backward recycling/passing.
- Commit message protocol now supports concise messages:
  - header + optional short bullets + `Patch-Notes:` line.

## Fixed

- Reduced cases where strikers recycle possession/back-pass instead of progressing toward goal in final-third situations.

- None.
